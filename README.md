# xrxs
OSINT framework in picolisp. 

Dependencies:
picolisp

Nice to have:
w3m
wget
nmap

Run 'pil x.l +' for interactive session with picolisp debugging at hand. 

csv.l and nmap.l can be used independently of xrxs.
